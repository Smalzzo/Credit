# Guide de captures d’écran

Ce dossier contient les captures d’écran non sensibles utilisées pour documenter le projet.

## Captures recommandées

1. `01_arborescence_repo.png`
   - Vue de l’explorateur VS Code montrant la structure du dépôt.

2. `02_api_health.png`
   - Réponse `GET /health` depuis navigateur, Swagger ou terminal.

3. `03_api_predict.png`
   - Exemple d’appel `POST /predict` avec réponse JSON.

4. `04_logs_jsonl.png`
   - Extrait de `data/production_logs.jsonl` (anonymisé, sans PII).

5. `05_streamlit_dashboard.png`
   - Dashboard Streamlit avec score distribution et métriques de latence.

6. `06_drift_report.png`
   - Aperçu de `reports/drift_report.html`.

7. `07_github_actions_ci.png`
   - Exécution du workflow CI sur GitHub Actions.

## Règles de sécurité

- Masquer toute donnée personnelle ou sensible.
- Ne pas exposer de token, clé API, URL privée, email personnel.
- Vérifier les métadonnées des captures avant publication.
